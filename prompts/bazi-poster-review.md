# 八字海报 · 评审—重生 Pass（v1.0，接在首遍 JSON 之后、render 之前）

> 首遍负责「出稿」，本遍负责「捞底」——只重写**质量不达标的字段**，达标的**逐字保留**。专治「同一张海报里字段参差不齐」。
> 分工：JSON 合法性、字段计数、禁词、句式、timeline 白名单、着色存在性由 `check-analysis.js` 脚本查（若已跑，其 FAIL 字段直接进重生名单）；**语义质量（锚定/落地/口吻/护栏）脚本看不见，全靠本遍**。
> 本遍是「挑错」任务：求保守、求一致、不求生动。

## 输入
1. `chart.txt`（算法层文本盘，唯一事实源）
2. 首遍产出的完整 `analysis.json`
3. Step0 选定的流派
4. （若有）`check-analysis.js` 的体检报告——其 FAIL 字段免评审直接重生

## 铁律
1. **抬地板，不抬天花板**：达标字段逐字不动；目标是把最差字段拉到及格线，不是把好句子改花。
2. **重生必须回 `chart.txt` 重新锚定**：重写任一字段，盘面依据（干支/十神/宫位/神煞/作用状态/用神出口）从 `chart.txt` 重取，不得沿用首遍可能编错的引证，不得新增文本盘以外的神煞或干支。
3. **不串派、不改排盘**；**状态照抄**（合冲刑害【生效/减力/虚拱…】不得把「减力」写成「有力」）。
4. **v2.3 适配**：`yongshen` 四格/divergence、`kaiyun` 用神/吉方/吉色/吉数/调候、大运流年配色均为**算法注入,不在评审范围**——只评 `yongshen.note_html` 与 `kaiyun` 的宜业/宜往/宜置/宜学/note_html。

## 第一步 · 逐字段评审（5 轴 + 类型红线）

**5 根通用轴（0=缺失 / 1=勉强 / 2=合格）**：任一红线踩中=FAIL；红线全过但 ≥2 轴 <2 = FAIL；否则 PASS。

| 轴 | 问自己 |
|---|---|
| A 锚定 | 有没有引具体盘面元素？还是换个盘也成立的通论？ |
| B 落地 | 每处结构证据后有没有**至少两倍篇幅**的现实意义？ |
| C 口吻 | 全程第二人称「你」，无「命主/日主为」播报腔，术语后跟白话？ |
| D 护栏 | 有没有断言可当场证伪的行为/事实（婚否/职业/收入/胖瘦/熬夜/爱不爱运动）？ |
| E 规格 | 满足该字段形态规格（下述红线）？ |

**类型红线**：
- **判词**：7字或4+4对仗、有画面感偏积极、不堆格局术语。
- **两句类**（geju.sub/wuxing.note/tg/yongshen.note）：恰两句，上句盘面、下句「所以你…」；`wuxing` 若出口含〔缺补说明〕未转述=FAIL；`yongshen.note` 边界盘未点明体用两分/分歧=FAIL。`dm.desc` 按日主固定句式（意象/特性是/意味着你/最强的能力/但）。
- **四大段落**：≥6~10句约200字起；**着色单位=完整特质短语**（零星单词或整段无色=FAIL）；`career` 财路叙事必须对照 chart.txt 十神结构（默认"才华生财"未核食伤生财链=FAIL）；`marriage` 必含正缘画像固定句式+整句标绿+紧跟"他/她会怎么对待你"，年龄择一或明确改用性格轴（骑墙=FAIL）。
- **三个精读段**：3~5句讲真正生效/最要紧的2~4项；状态照算法层；`yunsui` 只讲今年起5年+当前大运；顺逆措辞（出现「凶/大凶/灾年」=FAIL）；`shensha` 露出 via/tier/needs_review=FAIL、须按三段式（等级→主什么→意味着你什么加粗）。
- **短标签**（kaiyun.ye/place/item/skill、timeline[].desc）：具体不空泛、以出口开运用神/喜神为锚；timeline 年/岁必须取自「运岁引动→建议节点」白名单、重级节点若存在必须入选、措辞顺逆化。

产出一张**内部评审表**（不进最终 JSON）：逐字段 PASS/FAIL + FAIL 一句话病因（指名踩了哪条轴/红线）。

## 第二步 · 只重生 FAIL 字段
1. 回 `chart.txt` 取回该字段应有依据（1~3 条具体引证）；
2. 按类型规格重写，对症下药：缺锚补干支宫位；缺落地把意义写到两倍篇幅；踩护栏按句式转换表；骑墙择一；
3. 自查：病因消除？无新红线？未误改他字段？

**护栏句式转换表（照抄执行）**：
- ✗「你不太爱动」→ ✓「你的体质需要靠规律有氧把郁结代谢出去——**无论你现在动得多少**，这件事对你的回报都比常人高」
- ✗「你习惯熬夜」→ ✓「熬夜对你的损耗比常人更大，值得优先守住作息」
- ✗「你不善表达」→ ✓「你的表达欲偏内收，熟悉场合才充分释放」

## fail→fix 样例（照此力度）
**字段** `shensha.reading_html`
**首遍**：「你有天乙贵人、文昌、驿马、羊刃等神煞。天乙贵人是吉星，主贵人相助；驿马主奔波走动；羊刃较凶，要注意。tier 较高的几颗力量强。」

**重写**：「你盘中最亮的几颗星：**将星**坐年支，是一线的领导之星，<span class="hl-good">意味着你天生带镇场、带队的气场，站到台前比躲幕后更适合你</span>。**文昌贵人**会同天乙贵人齐聚月柱，主文采学业、逢难有人扶，<span class="hl-good">你在读书考试、文书专业进修这条线上有天分，关键处常遇贵人搭手</span>。**羊刃**坐日是把双刃剑，属警示类的刚烈之星，<span class="hl">你行事带狠劲、扛得住硬仗，但也容易过刚、和人硬顶，锋芒收放是一生的功课</span>。」

## 输出契约
**严格输出一份完整 `analysis.json`**（PASS 字段原样 + FAIL 字段重写后），直接以 `{` 开头 `}` 结尾，无解释、无 markdown 包装、无评审表。全字段 PASS 则原样回吐首遍 JSON。
