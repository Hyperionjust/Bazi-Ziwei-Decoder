## [1.2.0] · 2026-06-30 · 流派忠实度 & 文档重排
### Changed 调整
- **流派忠实度（解读层强化）**：选定流派后，模型**严格在该派 `core_techniques` 内、以该派文献（`literature`）为核心**分析，回避该派 `deemphasize` 判法，**不串派**；某结论该派文献无据则标存疑而非借他派——仅「不限 / open」才多派并陈。`bazi-prompt.md` 新增「流派忠实度」核心原则节，注入 `{{lineage.literature}}` / `{{lineage.deemphasize}}`；SKILL.md 同步约束。
- **文档**：README 框架重排——各模型安装说明收纳到「安装 / 使用指南」H2 下的子标题（H3），并新增顶部目录(TOC)。

## [1.0.0] · 2026-06-30 · 首个 Decoder 版（五层超级魔改）
> 基于 dzcmemory-web/bazi-ziwei-skill 重构发布，在其精准排盘内核之上叠加五层。
### Added 新增
- **神煞引擎**：24 神煞数据驱动起例（`shensha.json` + `shensha.ts`），写入算法层，带文献出处与 `needs_review` 标记。
- **流派镜片（6 派）**：子平 / 滴天髓 / 神峰通考 / 盲派 / 段氏(stub) / 不限；`--lineage` 仅换解读视角，**绝不改排盘**。
- **成长心态声明置顶**：每次八字分析开头必出（反「越算越不好」）。
- **输出模式 B**：总领速览 + 按需下钻（固定章号菜单、锚点防漂移、不诱导继续）。
- **文献核验底线**：起法分歧 / 查不到 → 标「⚠起法待核」，绝不编造。
- **回归测试**：`schema-check` 配置自检 + `fixtures` 神煞回归。
